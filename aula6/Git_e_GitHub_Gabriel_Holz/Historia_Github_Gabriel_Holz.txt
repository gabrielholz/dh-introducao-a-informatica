a. Quem inventou o sistema de controle de versão Git e por quê? 
  Criado por Linus Torvalds, Git é um software de controle de de versão desenvolvido para a eficiência e 
  confiabilidade da manutenção de aplicativo quando esses possuem um grande número de arquivos de código-fonte. 

b.Quem atualmente é o dono do Github e por quê? 
  Em 2018, a Microsoft, uma das principais parceiras da plataforma desde 2012, comprou o Github por US$ 7,5 bilhões.

c. Existe outra maneira, além do terminal, de trabalhar com o Github?
    Sim, além da linha de comando é possível gerenciar os projetos pelo site e também pelo aplicativo "Github Desktop".
